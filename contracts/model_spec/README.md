# Contracts Pack (Generated)

This folder contains generated contract packs derived from `docs/model_spec`.

Do not edit files under `contracts/model_spec/` by hand. Regenerate with:

```
python tools/build_contracts_pack.py --tag latest --force
```

The pack mirrors the contract files under `docs/model_spec/data-engine` and is intended
to provide a consolidated, read-only view for tooling.
